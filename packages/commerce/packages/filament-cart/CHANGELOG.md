# Changelog

All notable changes to `aiarmada/filament-cart` will be documented in this file.

## [0.1.0] - 2025-11-01

### Added
- Initial release
- Filament v4 admin plugin for cart management
- Cart resource with table and infolist views
- Cart item resource with nested display
- Cart condition resource for pricing rules
- Dashboard widget with cart statistics
- Bulk actions (clear, export)
- Dynamic condition management UI
- Real-time cart synchronization via events
- Normalized cart data for fast querying
- Price range and quantity filters
- Instance filtering (default, wishlist, quote)
- Comprehensive test suite with Pest v4
- Automatic resource discovery

### Dependencies
- PHP ^8.4
- Laravel ^12.0
- Filament ^5.0
- aiarmada/cart ^0.1
