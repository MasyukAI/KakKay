# Changelog

All notable changes to `aiarmada/docs` will be documented in this file.

## [0.1.0] - 2025-11-01

### Added
- Initial release
- Document generation package with PDF support
- Invoice generation with customizable templates
- Receipt generation
- Shipping label generation
- PDF rendering via DomPDF
- Template system with Blade views
- Document numbering and sequencing
- Multi-currency support
- Logo and branding customization
- Comprehensive test suite with Pest v4
- Template publishing for customization
- Automatic template caching

### Dependencies
- PHP ^8.4
- Laravel ^12.0
- barryvdh/laravel-dompdf ^3.0
