# Changelog

All notable changes to `aiarmada/filament-vouchers` will be documented in this file.

## [0.1.0] - 2025-11-01

### Added
- Initial release
- Filament v4 admin plugin for voucher management
- Voucher resource with create, edit, and view operations
- Usage history resource with redemption tracking
- Dashboard widget with voucher statistics
- Bulk actions (activate, deactivate, expire)
- Manual redemption workflow in admin panel
- Owner selector for multi-staff/multi-store scenarios
- Status filtering (active, expired, used up)
- Date range filtering
- Usage analytics and discount totals
- Optional Filament Cart integration for cart shortcuts
- Comprehensive test suite with Pest v4
- Configurable navigation and labels

### Dependencies
- PHP ^8.4
- Laravel ^12.0
- Filament ^4.0
- aiarmada/vouchers ^0.1
